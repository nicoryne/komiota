# Komiota - Web

This directory contains the web application for **Komiota**, serving as the central promotional hub and download portal for the mobile app.

## Overview

While the mobile app handles the heavy lifting of offline navigation and crowdsourcing, this landing page is designed to introduce Cebu City commuters to the Komiota platform. It provides a clean, fast, and accessible entry point for users to understand the app's mission and easily download it to their devices.

### Core Web Features

- **App Distribution:** Direct links to download the Komiota app for mobile devices.
- **Feature Showcase:** A brief overview of the app's offline capabilities and the goal of mapping out the new CBRT system.
- **Responsive Design:** Highly optimized for mobile browsers so commuters can easily read about the app and access download links while on the go.

## Tech Stack

- **Framework:** Next.js 16.2.1
