## About The Project

With the recent rollout of the Cebu Bus Rapid Transit (CBRT) system around Cebu City, many commuters are left confused about where the new bus stops are located. Because this infrastructure is so new, online maps haven't caught up yet.

Komiota (a play on the Cebuano phrase "Commute Ta") is a community-driven application designed to bridge that gap. It maps out the new transit routes and bus stops, ensuring commuters can easily navigate the city even without an active internet connection.

### Key Features

- **Offline-First Navigation:** Access routes and locate bus stops anytime, anywhere, without needing mobile data.
- **Interactive Map:** View verified transit stops and routes overlaid on an interactive map.
- **Community Crowdsourcing:** Users can upload new or missing bus stops—complete with images and location data—which are then verified and synced to the rest of the community.

## Repository Structure

This project is set up as a monorepo containing both the main application and its companion website.

- `/mobile` — The mobile application codebase.
- `/web` — The codebase for the Komiota website and promotional landing page.
